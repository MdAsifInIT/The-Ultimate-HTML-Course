There are three types of lists - Ordered, Unordered and Definition lists.

The following are a list of programming languages in ordered list:

  1. C
  2. C++
  3. Java
  4. Python
  5. JavaScript

The following are the file extensions for the languages in unordered list:

  • .c
  • .cpp
  • .java
  • .py
  • .js

The Following are the definitions of the languages in definition list:

C
A powerful, low-level programming language used for system programming and creating operating systems due to its direct memory access and efficient performance.
C++
An extension of C that supports object-oriented programming, widely used for software development, game development, and performance-critical applications.
Java
A high-level, object-oriented language designed for portability across platforms ("write once, run anywhere") and used for building large-scale applications.
Python
A high-level, versatile language known for its simple syntax and readability, commonly used in web development, data science, automation, and AI.
JavaScript
A dynamic, high-level language primarily used for adding interactivity to websites and building web applications, both on the client and server side (with Node.js).
